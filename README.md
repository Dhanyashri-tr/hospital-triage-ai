# 🏥 AI Hospital Triage System

This project simulates an AI-powered hospital triage system that prioritizes patients based on symptoms and vitals.

## 🚀 Features
- Patient triage classification (TREAT_NOW, MONITOR, WAIT)
- Priority scoring
- Simulated hospital environment
- AI-based reasoning
