# 🏥 Hospital Triage AI

This is an AI-powered hospital triage system built using FastAPI and OpenEnv.

## 🚀 Features

- Patient triage classification
- Priority scoring
- Step-based environment simulation
- API endpoints for integration

## 📡 API Endpoints

- `GET /` → Health check
- `POST /reset` → Reset environment
- `POST /step` → Perform triage step

## ⚙️ Tech Stack

- FastAPI
- OpenEnv
- Docker
